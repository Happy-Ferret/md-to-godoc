# md-to-godoc

Sort of like [godocdown](https://github.com/robertkrimen/godocdown), but in
reverse.

md-to-godoc takes markdown as input, and generates godoc-formatted package
documentation.

## Status

Way, way alpha. Barebones. The minimalest.
